# mlops-engineering-journey
A 100-day hands-on MLOps journey covering Python, DVC, MLflow, Docker, Kubernetes, CI/CD, Argo Workflows, Kubeflow, Monitoring, and Production ML Systems.

## 📚 Learning Roadmap & Technologies

| Days       | Focus Area                   | Technologies                                                                             |
| ---------- | ---------------------------- | ---------------------------------------------------------------------------------------- |
| Day 1–10   | Python & Foundations         | Python, pip, venv, Jupyter Notebook, UV, Makefile, Ruff, Black, Pre-Commit, Cookiecutter |
| Day 11–19  | DVC                          | DVC, Git, Local & Remote Storage                                                         |
| Day 20–30  | MLflow                       | MLflow Tracking, Model Registry, Artifact Management                                     |
| Day 31–40  | Training & Optimization      | Scikit-Learn, PyTorch, Optuna, FLAML, YAML                                               |
| Day 41–49  | Feature Store & Data Quality | Feast, HashiCorp Vault, Great Expectations                                               |
| Day 50–66  | Docker & Model Serving       | Docker, Docker Compose, FastAPI, Flask, BentoML                                          |
| Day 67–75  | Monitoring & Observability   | Prometheus, Grafana, Evidently AI                                                        |
| Day 76–84  | CI/CD for ML                 | GitHub Actions, CML, Automated Testing & Validation                                      |
| Day 85–96  | Argo & Kubernetes            | Argo Workflows, ArgoCD, Kubernetes, KServe, Kubeflow                                     |
| Day 97–100 | End-to-End MLOps Capstone    | Complete Production MLOps Platform                                                       |
